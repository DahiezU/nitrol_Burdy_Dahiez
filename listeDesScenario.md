# Liste des scénarios d'application :
1. **Authentification :** 
    - S'inscrire  
    - Se connecter/ déconnecter
    - Modifier profil
    - Supprimer profil
2. **Escpace Bibliothèque :** 
    - Enprunter oeuvre
    - Louer oeuvre
    - Rechercher oeuvre
    - Numériser oeuvre
    - Proposer oeuvre 
3. **Moderation :** 
    - Compléter données l'oeuvre
    - Modifier données oeuvre
    - Supprimer oeuvre
    - Decider statut de l'oeuvre
